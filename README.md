# Mini QHSE IA

Mini SaaS QHSE avec intelligence artificielle et paiement mobile Moneroo pour le marché africain.

## Stack Technique

- **Frontend/Backend**: Next.js 14 (App Router)
- **Base de données**: PostgreSQL (Supabase)
- **Auth**: NextAuth.js
- **IA**: Claude API (Anthropic)
- **Paiement**: Moneroo.io (Mobile Money)
- **UI**: Tailwind CSS + shadcn/ui

## Développement

```bash
npm install
npm run dev
```

## Déploiement

```bash
docker-compose up -d
```

## Structure

- `/src/app` - Routes Next.js (frontend + API)
- `/src/components` - Composants React
- `/src/lib` - Utilitaires et clients
- `/prisma` - Schéma base de données
